Cytoscape: Linux Source Install and Build at the ISB

There are 5 steps you must follow in order to install and build cytoscape and its plugins, from source, at the ISB.
  1. setup CVS

    • ssh isb@ideker2.wi.mit.edu -L 2401:localhost:2402 (see Paul for password)
    • set the environment variable CVSROOT to this value:

      :pserver:isb@localhost:/common/cvsdir2

  2. do a CVS checkout

    • cd yourWorkingDirectoryOfChoice
    • cvs login (same password as above)
    • cvs co cytoscape
    • cvs co csplugins

  3. Set up your environment.

    At a minimum you will need:

    • java 1.4 in your path
    • a java CLASSPATH which includes
      1. cytoscape/aux.jar
      2. /net/compbio/src/distributions/yfiles1.4/yfiles/y.jar
      3. /net/compbio/jars/junit.jar

    If you want to use the all-pairs-shortest-path calculations, in the APSP plugin, add this your LD_LIBRARY_PATH:

    • /net/compbio/src/distributions/yfiles1.4/LEDAGraphAlgo-1.0.1/linux

    If you wish to build C++ plugins that use LEDA, you will also need

    • LEDA_HOME=/net/compbio/src/distributions/LEDA-4.1
    • CPPUNIT_HOME=/net/compbio
    • add to your LD_LIBRARY_PATH
      • /net/compbio/src/distributions/LEDA-4.1
      • /net/compbio/lib

  4. Build the cytoscape packages from source

    • cd cytoscape
    • make
    • make test

  5. Build the csplugin packages from source

    • cd csplugins
    • make
    • make test

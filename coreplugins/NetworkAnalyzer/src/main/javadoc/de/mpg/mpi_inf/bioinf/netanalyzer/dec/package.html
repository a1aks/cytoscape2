Definitions of chart decorators.

A decorator is a class that provides additional information on a chart. One decorator can be applied to a chart of one or more complex parameter instances. In its definition in XML settings file, every complex parameter includes information about which decorators can be applied to it. A decorator is activated (applied) by the user by clicking on its button in the chart panel.

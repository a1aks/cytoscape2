Labels for Physical Entities
All physical entities will be labeled as follows:
[NAME] [-CHEMICAL MODIFICATION ABBR] [(CELLULAR LOCATION ABBR)]
where [CHEMICAL MODIFICATION ABBR] and [CELLULAR LOCATION ABBR] are one of the abbreviations described below.

Examples
MITF MITF
MITF Phosphorylated MITF-P
MITF Phosphorylated, in the cytoplasm MITF-P (CP)

How to use the cytoscape.props file

  • LOGGING
    specify which diagnostic messages from the program are visible on the user's console.

    logging=<value>

    where <value> may be one of

    1. severe (the default)
    2. warning
    3. info
    4. config
    5. all
    6. off
    7. none (same as off )


  • NODE SHAPE specify which node attribute controls node shape, and what the mapping should be. 

    node.shape.controller=<attributeName>
node.shape.categorizer.type=discrete
node.shape.categorizer.map.true=<value>

    where <attributeName> may be the name of any attributed defined on nodes in your graph and
    where <value> may be one of

    1. circle
    2. square

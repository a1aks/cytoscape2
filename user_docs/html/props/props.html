How to use the cytoscape.props file

  • LOGGING
    specify which diagnostic messages from the program are visible on the user's console.

    logging=<value>

    where <value> may be one of

    1. all
    2. severe (the default)
    3. warning
    4. info
    5. config
    6. off
    7. none (same as off )

    Lower values imply all higher values. For instance, when you specify logging=info you will see not only info messages, but also warning and severe messages.

  • NODE SHAPE specify which node attribute controls node shape, and what the mapping should be. 

    node.shape.controller=<attributeName>
node.shape.categorizer.type=discrete
node.shape.categorizer.map.<attributeValue>=<shape>

    where

    1. <attributeName> may be the name of any attribute defined on nodes in your graph;
    2. <attributeValue> is any value it may take on; and
    3. <shape> may be circle, square, or ...
